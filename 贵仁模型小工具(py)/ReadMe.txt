贵仁模型小工具（py）：


/* ----------------- 版本转换工具 ----------------- */
1. version_convert_1to2.py，将1.2.6版本的一维模型的主要部分转为2.0版本。

2. version_Convert.cpp/h， 将差分格式的一维模型转为有限体积格式版本。



/* ----------------- 断面处理工具 ----------------- */
1. HD_processing.py， 关于HD数据的前后处理工具。

2. test.py， 对工具的单元测试（unittest模块）。



/* ----------------- 与SMS转换工具 ----------------- */
1. MeshToGeo.h/cpp，SMS软件的geo文件与1.2.6版本的二维模型输入文件的互转。



/* ----------------- 与SWMM转换工具 ----------------- */
1. grms2swmm.h/cpp，variable.h，将贵仁1.2.6版本的一维模型转为SWMM软件格式。

2. SwmmToGrms.h/cpp，text.h，vars.h， 将SWMM软件模型转为贵仁1.2.6版本的格式。
